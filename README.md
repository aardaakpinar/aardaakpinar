## 🧭 My Journey

My journey in software development started around **2019–2020**, when the COVID-19 pandemic forced everyone to stay at home. Inspired by the thought, *“If they can do it, so can I,”* I built my first application, **NAR** a project where I tried to combine everything I was experimenting with at the time.

So far, I’ve worked on many projects some conceptual, some live, and some eventually discontinued. Each one taught me something new and helped shape how I approach problems and build software.
I enjoy **building tools and applications that challenge me**, experimenting across different domains, and sharing my journey with the community.

## 📂 Projects

* **Pitopi** *(May 2025 – Present)*
  Real-time peer-to-peer chat application with modern UI, file sharing, and story-style content system. Focused on decentralized messaging.
  [Project Link](https://github.com/pitopichat/pitopi)

* **Limon** *(Apr 2020 – Present)*
  CLI-based chatbot for personal projects.
  [Project Link](https://github.com/aardaakpinar/limon)

* **NaC (Not a C)** *(Oct 2025 – Mar 2026)*
  Interpreted scripting language in C for teaching programming and rapid prototyping. Minimal syntax, basic types, control structures, and I/O.
  [Project Link](https://github.com/naclang/nac)

* **Bi’Bak – RSS Reader** *(Feb 2020 – Feb 2026)*
  Client-side RSS/Atom reader with CORS handling, IndexedDB storage, bookmarking, OPML import, and real-time updates.
  [Project Link](https://github.com/aardaakpinar/bibak)

* **Kargo** *(Feb 2026 – Feb 2026)*
  Lightweight, keyboard-focused web browser built with Electron. Minimal interface and cross-platform support.
  [Project Link](https://github.com/aardaakpinar/kargo)

* **SayıDışı** *(May 2024 – Feb 2026)*
  Logic-based number game where players find missing digits.
  [Project Link](https://github.com/aardaakpinar/sayidisi)

* **INH (I'm Not Hacker)** *(Jun 2024 – Oct 2025)*
  CMD-style website aiming to create a social network for sharing programs.

* **Pigeongo** *(Jun 2024 – Aug 2025)*

  [Project Link](https://github.com/aardaakpinar/pigeongo)

* **MASA / Fireverse** *(Aug 2021 – Jun 2024)*
