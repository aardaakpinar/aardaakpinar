## üß≠ My Journey

My journey in software development started around **2019‚Äì2020**, right when the COVID-19 pandemic forced everyone to stay at home. Inspired by the thought, _"If they can do it, so can I,"_ I built my first application called **NAR** a name I chose because I tried to integrate everything I experimented with into it.

After NAR, I decided to move into more professional domains. My next project was a social media app named **Masa**, which I developed until **2024**.
